# Scam Detector Python

A beginner Python project that detects potentially scam-related job or internship messages using keyword matching and basic pattern detection.

## Features
- Detects scam-related keywords in text
- Flags suspicious payment-related phrases
- Detects phone numbers in job/internship messages
- Generates a simple risk score (Low / Suspicious / High Risk)

## How to run
```bash
python main.py
